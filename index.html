Poziom Advanced

Celem tego modułu jest zapoznanie się z zaawansowanymi możliwościami języka JavaScript. Zostanie omówiony standard EcmaScript 6, biblioteka jQuery, AJAX oraz jQuery UI. Uczestnicy kursu poznają też różne przykłady API oraz jak obsługiwać i debugować błędy w języku JavaScript.
